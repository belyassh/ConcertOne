# ConcertOne

Универсальный шаблон лендинга для артиста, тура или фестиваля с блоками продажи билетов.

## Что внутри

- `index.html` - структура страницы
- `styles.css` - визуальный стиль и адаптив
- `script.js` - конфиг артиста, таймер, мобильное меню, анимации

## Быстрый старт

1. Открой `index.html` в браузере.
2. Измени данные в объекте `SITE_CONFIG` в `script.js`.
3. Поставь реальную ссылку на билеты в поле `ticketUrl`.

## Кастомизация под любого артиста

Минимальный набор полей в `script.js`:

- `artist`
- `guest`
- `city`
- `venue`
- `eventDateISO`
- `eventDateLabel`
- `ticketUrl`

Все элементы на странице, где есть атрибуты `data-bind` и `data-bind-href`, обновятся автоматически.
